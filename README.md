# Proyecto TelCo

Aplicación web desarrollada con PHP y CodeIgniter 4 que simula una plataforma de gestión de usuarios y productos de software agricola.

## 🚀 Tecnologías utilizadas

- 🔹 PHP 8.x
- 🔹 CodeIgniter 4.x
- 🔹 MySQL
- 🔹 Bootstrap 5
- 🔹 HTML5, CSS3
- 🔹 JavaScript
